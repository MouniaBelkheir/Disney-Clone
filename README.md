# The Movie  
---
## Description

The Movie is a movie app built using React, Vite, Tailwind CSS, and The Movie Database (TMDB) API. It allows users to search for movies and view details about them.

## Technologies Used

- React
- Vite
- Tailwind CSS
- The Movie Database (TMDB) API
- React Slick
